# epochadd

add year, month, day to unix epoch timestamp

# download
`curl https://raw.githubusercontent.com/Luke31/epochadd/main/epochadd --output ~/bin/epochadd`

# usage
```
epochadd <inputepoch> <years> <months> <days>
```
